# 컴퓨터 기초 개념

## 컴퓨터 언어

컴퓨터 언어는 사람과 컴퓨터가 의사소통하기 위한 수단이다.

컴퓨터는 다음을 모두 이진법으로 처리한다.

- 저장

- 연산

- 논리 처리

### 이진법을 사용하는 이유

컴퓨터는 전기 신호 기반으로 동작한다.

전기는 두 상태로 표현된다.

- 흐름 있음 → 1

- 흐름 없음 → 0

따라서 2진법을 사용한다.

# Java 기초

## Java

Java는 다양한 플랫폼에서 실행될 수 있도록 설계된 프로그래밍 언어이다.

### 특징

- 플랫폼 독립성

- 보안 안정성

- 자동 메모리 관리

- 객체지향

---

## 실행 구조

### C

소스코드 → 기계어

특징

- 빠르다

- 하드웨어 직접 제어 가능

- 플랫폼 종속적

### Java

소스코드 → 바이트코드 → JVM → CPU

특징

- 한번 작성하면 여러 환경에서 실행 가능

# JDK / JRE / JVM

## JDK

자바 개발에 필요한 도구 모음이다.

구성:

- JRE

- 컴파일러

- 디버거

## JRE

자바 실행을 위한 최소 환경이다.

구성:

- JVM

- 클래스 라이브러리

## JVM

바이트코드를 실행하는 엔진이다.

역할:

- 기계어 변환

- 메모리 관리

- 프로그램 실행

# 스택과 힙

## 스택

실행 흐름 저장 공간

### 특징

- 메서드 실행 시 생성

- 종료 시 자동 반환

- 빠른 할당/해제

- 스레드별 독립 공간

### 비유

개인 작업 공간

---

## 힙

객체 저장 공간

### 특징

- 객체 저장

- 여러 실행 흐름 공유

- 장기 저장 데이터 관리

### 비유

공용 저장 공간

---

# GC (Garbage Collector)

힙 메모리를 자동으로 관리하는 기능이다.

## 역할

- 참조되지 않는 객체 제거

- 메모리 정리

- 메모리 압축

## 기준

스택 어디에서도 참조하지 않는 객체

## 특징

- 자동 메모리 정리

- 실행 중 잠시 멈출 수 있음

# 기본형 / 참조형 타입

## 기본형 타입

실제 값 자체를 저장한다.

종류:

- boolean

- char

- byte

- short

- int

- long

- float

- double

특징:

- 값 저장

- 주로 스택 사용

---

## 참조형 타입

- 객체 주소를 저장한다.

### 예:
- String
- 배열
- 객체

특징:

- 실제 데이터는 힙

- 변수는 주소 저장

---

# String

문자열을 저장하는 참조형 타입이다.

특징:

- 문자열 객체는 힙에 저장된다.

- 변수에는 객체 주소가 저장된다.

- 한번 생성되면 수정할 수 없다.

- 값 변경 시 새로운 객체가 생성된다.

- 불변 객체이다.

---

# String Constant Pool

- 동일한 문자열을 재사용하기 위한 공간이다.

특징:

- 동일 문자열은 중복 생성하지 않는다.
- 기존 객체를 공유한다.
- 메모리 효율을 높인다.
